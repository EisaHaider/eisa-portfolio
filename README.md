# Eisa Haider | Portfolio & Professional Platform

A bilingual personal portfolio and professional platform built with Python, Flask, and Tailwind CSS. The website serves as a central hub for showcasing professional experience, selected projects, technical interests, and ongoing initiatives.

## Purpose

This platform provides a structured overview of:

* **Information Systems & Software Development** – Experience in designing, developing, and maintaining business applications and information systems.
* **Selected Projects & Initiatives** – Highlighting major projects such as Document Management Systems (DMS), Electronic Medical Records (EMR), workflow automation initiatives, and custom business applications.
* **Professional Domains** – Covering enterprise information systems, document management solutions, workflow automation, cloud-native solutions, and digital skills development.
* **Areas of Interest** – Exploring emerging technologies, user experience, running, and health & nutrition.

## Features

* **Bilingual Interface (English & Arabic)** with full LTR/RTL support.
* **Light and Dark Modes** with persistent user preferences.
* **Responsive Design** optimized for desktop, tablet, and mobile devices.
* **Client-Side Internationalization (i18n)** powered by JavaScript.
* **Scroll-Based Animations** using the Intersection Observer API.
* **Tailwind CSS Build Pipeline** for efficient styling and asset management.
* **Lucide Icons Integration** for lightweight and consistent iconography.

## Local Development

### Prerequisites

Ensure the following software is installed:

* Python 3.x
* Node.js
* npm

### Build CSS Assets

Start the Tailwind CSS watcher:

```bash
npm run watch-css
```

### Run the Development Server

```bash
flask run
```

## Deployment

The project is designed for deployment on cloud platforms such as:

* PythonAnywhere
* Google Cloud Run
* Docker-based environments
